# 🐍 Snake Adventure JS

A classic Snake Game built using **HTML, CSS, and JavaScript** with a modern UI, responsive design, score tracking, timer, and persistent high score using Local Storage.

## 🎮 Features

- 🐍 Smooth snake movement
- 🍎 Random food generation
- 📈 Score tracking
- 🏆 High score saved using Local Storage
- ⏱️ Live game timer
- 🎯 Keyboard Controls
  - Arrow Keys
  - W A S D
- 💀 Game Over screen
- 🔄 Restart Game
- 📱 Responsive Design

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript 

## 🎮 Controls

| Key     | Action     |
|-------- |------------|
| ⬆️ / W  | Move Up    |
| ⬇️ / S  | Move Down  |
| ⬅️ / A  | Move Left  |
| ➡️ / D  | Move Right |


## 🛠️ Installation

Clone the repository:

```bash
git clone https://github.com/jayanthvd25/Snake-Adventure-JS.git
```

Open the project:

```bash
cd Snake-Adventure-JS
```

Run the project by opening `index.html` in your browser.
